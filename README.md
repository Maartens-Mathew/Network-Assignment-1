# Network Assignment 1 - Encrypted Chat Protocol

A university project for CSC4026Z Network Security that implements a secure chat protocol with WireGuard encryption.

## Overview

This project develops a chat protocol for communication with a remote server, leveraging WireGuard for transport-layer encryption. The application is built in Python and utilizes a modern Python package manager for dependency management.

## Features

- **Secure Chat Protocol**: Custom implementation of a chat protocol
- **WireGuard Encryption**: End-to-end encryption using the WireGuard protocol
- **Remote Server Communication**: Designed for client-server architecture
- **Python-based**: Implemented entirely in Python for flexibility and ease of development

## Project Status

- [ ] Core chat protocol implementation
- [ ] WireGuard integration
- [ ] User interface component (planned)

## Requirements

- Python 3.8 or higher
- `uv` package manager
